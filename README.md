# BASD: Bias-Aligned Spectral Distillation

Knowledge distillation for Vision Transformers is difficult because teacher and student models often differ in token count, embedding dimension, attention heads, and depth, and existing methods rely on hand-tuned projectors and fixed layer mappings that fail to generalize across architectures. BASD is an architecture-agnostic distillation framework that transfers inductive bias through token geometry by operating on four Riemannian manifolds. A Grassmannian-based layer selector computes subspace distances between teacher and student representations to produce soft mixing weights over all teacher layers, with subspace rank determined automatically using Marchenko-Pastur theory. Cross-attention projectors constrained to the Stiefel manifold align teacher tokens to the student spatial grid while preserving inner-product structure via polar retraction after each update. The distillation objective combines three components: redundancy suppression using Barlow Twins cross-correlation, dual-space geometric transfer via attention-weighted Procrustes alignment in sample space and Bures-Wasserstein matching in feature space, and attention pattern transfer through KL divergence with learned head alignment and temperature scaling. An inverse-loss softmax scheme automatically balances these terms using a single temperature hyperparameter. BASD unifies ViT-to-ViT and CNN-to-ViT distillation under the same interface, treating CNN features as single-layer tokens with synthetic uniform attention and processing both teacher types identically.
