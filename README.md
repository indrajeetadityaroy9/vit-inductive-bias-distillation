# BASD: Bias-Aligned Spectral Distillation

Knowledge distillation for Vision Transformers is difficult because teacher and student models often differ in token count, embedding dimension, attention heads, and depth, and existing methods rely on hand-tuned projectors and fixed layer mappings that fail to generalize across architectures. BASD is an architecture-agnostic distillation framework that transfers inductive bias through token geometry using three mathematically orthogonal objectives. A Grassmannian-based layer selector computes subspace distances between teacher and student representations to produce soft mixing weights over all teacher layers, with per-layer subspace rank determined automatically using Marchenko-Pastur theory. Cross-attention projectors constrained to the Stiefel manifold align teacher tokens to the student spatial grid while preserving inner-product structure via SVD retraction after each update. The distillation objective combines attention-weighted Procrustes alignment on the token cloud (handling mismatched student and teacher dimensions natively through rectangular SVD of the cross-covariance), redundancy suppression via Barlow Twins cross-correlation driving the student-teacher correspondence matrix toward identity, and attention pattern transfer through KL divergence with learned head alignment and temperature scaling. An inverse-loss softmax scheme (UW-SO; Kirchdorfer et al. 2024) automatically balances all components including cross-entropy. BASD unifies ViT-to-ViT and CNN-to-ViT distillation under the same interface, treating CNN features as single-layer tokens with synthetic uniform attention and processing both teacher types identically.
